    The many views of a Software Carpentry workshop

    Will Furnass

    2017-09-14 18:01

    RSE Sheffield recently ran our first Software Carpentry workshop. Over two days in the shiny new Diamond building we introduced thirty researchers to:

    • The Unix Shell, including the basics of automation
    • Programming with Python (with a data science perspective)
    • Version Control with Git
    • Using Databases and SQL

    We knew there would be demand for the workshop given the topics covered and it was free to attend but were still surprised when all thirty EventBrite places were snapped up in twenty minutes and the waiting list grew to sixty names.
